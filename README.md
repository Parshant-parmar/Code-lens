# 🚀 CodeLens

AI-powered code analysis platform built with Node.js, Express, and Google Gemini.

## Features

- AI code review
- Time & Space Complexity estimation
- Optimization suggestions
- Bug detection
- Refactored code
- Interview-style feedback

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Google Gemini API

## Run Locally

```bash
npm install
npm run dev
```

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```
